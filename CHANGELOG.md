# Changelog — EVEMON SMP Website

Semua perubahan penting pada proyek website resmi **EVEMON SMP** akan dicatat dalam file ini secara berkesinambungan.

## [1.0.0] — 2026-07-07

### Added
- **Peluncuran Perdana Website Resmi EVEMON SMP** dengan desain *Cinematic Dark Luxury*.
- **Integrasi Developer Bar Dinamis** yang melakukan fetch data profil pengembang dari GitHub secara real-time.
- **Header Navigasi Sticky** dengan efek blur, transisi solid, dan burger menu yang dinamis untuk perangkat mobile.
- **Section Filosofi Editorial** dengan tampilan tulisan bersahaja dalam ruang kosong yang luas.
- **Section Informasi Server** yang menampilkan IP Java, Bedrock, Port, serta tombol pintasan "Salin IP" instan.
- **Section Regulasi/Aturan** berupa kartu modern dengan ikon adaptif Lucide React.
- **Section Konsekuensi Premium** berupa layout kutipan (*block-quote*) besar berestetika tinggi.
- **Footer Minimalis** yang memuat lisensi hak cipta dan jembatan tautan pengembang eksternal.
- Dokumen pendukung repositori open source professional (`README.md`, `LICENSE`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`, `.gitattributes`, `.editorconfig`, `vercel.json`, templates issue & PR).
